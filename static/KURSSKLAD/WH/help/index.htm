1.     Общие положения

1.1.            Настоящий документ разработан для описания экранной формы и функциональности  интерфейса «Редактор склада», взаимосвязей этого интерфейса с процессами и другими интерфейсами, возможных неисправностей и путей их устранения.

1.2.            Интерфейс «Редактор склада» представляет собой модуль  RBS-WMS, который предназначен для прорисовки склада.

1.3.            В рамках описываемого интерфейса выполняются следующие задачи:

-               редактирование существующих прорисовок;

-               создание новых прорисовок.

1.4.            Интерфейс «Редактор склада» находится в системе RBS – Склад.

2.             Описание функциональности

3.1.            Работа в интерфейсе «Редактор склада»: для того, чтобы начать работу в интерфейсе «Редактор склада» необходимо авторизироваться и в меню RBS-WMS выбрать «Редактор склада» (рис. 13) и выбрать зону работу (рис. 3).

Рис.13

3.2.            Изменить масштаб прорисовки: для того чтобы изменить масштаб прорисовки склада, увеличьте либо уменьшите размер на панели меню (рис. 1).

3.3.            Редактировать склад: для того чтобы редактировать уже существующий (прорисованный) склад, на панели меню выберите «Выбор склада» (рис. 1). В открывшемся окне (рис. 3) выберите необходимый склад, перетянув его название на область прорисовки.

3.3.1.      Для перемещения объекта склада (стеллажа, конвейера, ворот и т.д.) необходимо перетянуть объект, зажав левую кнопку мыши. Для перемещения ряда, сначала перетяните любой из стеллажей ряда, потом, зажав левой кнопкой мыши ряд в том месте, откуда был перемещен стеллаж, перетяните ряд.

3.3.2.      Для того чтобы изменить размеры объекта (увеличить или уменьшить) подведите курсор к одной из сторон объекта, после того, как курсор примет форму двухсторонней стрелки растяните или сожмите выбранную сторону до необходимого размера.

3.4.            Изменить объект: для того чтобы изменить объект, вызовите контекстное меню, нажав левой кнопкой мыши на необходимый объект, и выберите «Изменить» (рис. 5). В появившемся окне «Изменение элемента» (рис. 9) измените необходимые поля и нажмите на кнопку «Подтвердить».

3.5.            Копировать ряд: для того чтобы скопировать ряд, вызовите контекстное меню, нажав левой кнопкой мыши на необходимый объект, и выберите «Копировать» (рис. 5). Затем снова вызовите контекстное меню и выберите команду «Вставить». В появившемся окне «Копирования ряда» (рис. 10) введите название объекта и нажмите кнопку «Подтвердить».

3.6.            Удалить объект: для того чтобы удалить объект, вызовите контекстное меню, нажав левой кнопкой мыши на необходимый объект, и выберите команду «Удалить» (рис. 5). В появившемся окне подтверждения (рис. 11) нажмите на кнопку «ОК» для подтверждения осуществляемого действия.

3.6.1.           Для того чтобы удалить объект, перетяните его в «Корзину» панели меню (рис. 1).

3.7.            Скрыть объект: для того, чтобы скрыть объект (объект существует в реальности, но не отображается на прорисовке склада), вызовите контекстное меню, нажав левой кнопкой мыши на необходимый объект, и выберите команду «Скрыть» (рис. 5). В появившемся окне подтверждения (рис. 11) нажмите на кнопку «ОК» для подтверждения осуществляемого действия.

3.8.            Показать скрытые объекты: для просмотра скрытых объектов, вызовите контекстное меню и выберите команду «Скрытые» (рис. 5).

3.8.1.      Для того чтобы скрытый объект отображались в области прорисовки, выполните действия п. 3.7. Из появившегося окна «Без координат» (рис. 12) перетяните необходимый объект в область прорисовки.

3.9.       Очистить область прорисовки: для того чтобы очистить область прорисовки, вызовите контекстное меню и выберите команду «Очистить» (рис. 5).

3.10.             Добавить объект: для того чтобы добавить новый объект, на панели меню выберите «Типы местоположений». В открывшемся окне (рис. 4) выберите необходимый тип объекта, перетянув его название на область прорисовки. В окне «Добавления элемента» (рис. 6, рис. 7) заполните поля  и нажмите кнопку «Подтвердить».

3.11.             Сохранить изменения: для того чтобы сохранить изменения, выберите «Сохранить» на панели меню (рис. 1).

4.                  Взаимодействие с процессами и другими интерфейсами    

4.1.       Интерфейс «Редактор склада» взаимодействует с интерфейсом «Мониторинг склада», предоставляю актуальные данные и прорисовку склада.
