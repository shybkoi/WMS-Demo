Термины и сокращения

ШК – последовательность чёрных и белых полос, представляющая некоторую информацию в удобном для считывания техническими средствами виде

ТСД – терминал сбора данных - переносное устройство в виде карманного компьютера, оснащенного сканером штрих-кодов, которое служит для ускорения операций по учету и обработке товаров

 

1.                  Общие положения

1.1.            Настоящий документ разработан для описания экранной формы и функциональности  интерфейса «Изменение зоны», взаимосвязей этого интерфейса с процессами и другими интерфейсами, возможных неисправностей и путей их устранения.

1.2.            Интерфейс «Изменение зоны» представляет собой модуль  RBS-WMS, который предназначен для изменения зоны работы ТСД.

1.3.            В рамках описываемого интерфейса выполняется задача – изменение зоны работы сотрудника.

1.4.            Интерфейс «Изменение зоны» находится в подсистеме RBS – Терминал и в главном меню ТСД.

2.                  Описание экранных форм

2.1.            Интерфейс «Изменение зоны» состоит из экранной формы (рис. 1), в которой отображается ФИО авторизованного сотрудника и список возможных зон действия ТСД с указанием статуса работы ТСД в этой зоне («+» и «-»).

Рис. 1

3.             Описание функциональности

3.1.            Работа в интерфейсе «Изменение зоны»: для того, чтобы начать работу в интерфейсе «Изменение зоны» необходимо авторизоваться и в меню ТСД выбрать «Изменение зоны» (рис. 2).

Рис. 2

3.2.            Изменить зону: для того чтобы изменить зону работы ТСД нажмите на необходимую вам зону. Напротив зоны появится «+» (рис. 1).

4.                  Взаимодействие с процессами и другими интерфейсами

4.1.            Интерфейс «Изменение зоны» взаимодействует со всеми терминальными интерфейсами, если есть дробление объекта на зоны.
